Développement Web

Le développement web regroupe divers métiers :

  • Front-End : crée l'interface visible (HTML, CSS, JS)
  • Back-End : gère la logique serveur et les bases de données
  • Full-Stack : maîtrise le front-end et le back-end
  • DevOps : assure le déploiement et la stabilité des applis
  • Mobile : conçoit des applis pour smartphones
  • Blockchain : La conception, le développement, et la mise en œuvre de solutions décentralisées
